READ ME
=======

Run python demo.py on terminal.
Input image used is 2007_009654.jpg and can be changed by changing img variable in python code.
It uses gist.py to get global features, which is a descriptor of length 960.

Requirements
============
Python 3
Numpy
OpenCV
